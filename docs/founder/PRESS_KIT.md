# Shipwright Press Kit

## One-Line Description
Shipwright turns dormant GitHub repos into live products in minutes, using AI to generate deployment configs, READMEs, landing pages, and pull requests.

## Short Description (100 words)
Shipwright is a developer tool that analyzes any GitHub repository and generates everything needed to deploy it: a Vercel configuration, professional README, landing page, and environment variable template — all packaged into a single pull request. Built for solo developers and indie builders who have projects sitting undeployed on GitHub. Sign in with GitHub, pick a repo, and get a production-ready PR in under 30 seconds. Pricing starts at $5 per generation or $15/month for unlimited access.

## Key Features
1. **Framework detection** — automatically identifies Next.js, React, Vue, Express, etc.
2. **Risk scoring** — rates deployment readiness on a 0–100 scale
3. **AI generation** — README, landing page copy, and env templates via Gemini 1.5 Flash
4. **PR automation** — all generated files packaged into a single GitHub pull request
5. **Vercel-native** — generates `vercel.json` optimized for the detected framework

## Story / Angle for Journalists
**"The deployment tax":** Most indie developers build 80% of a project then abandon it because the remaining 20% — deployment config, documentation, landing page — feels like a different job entirely. Shipwright eliminates that tax by automating the boring-but-necessary parts of shipping.

**Indie dev market:** There are an estimated 100M+ GitHub repositories, with a large percentage never deployed. Shipwright targets the segment of active developers (5-10M) who regularly build but infrequently ship.
