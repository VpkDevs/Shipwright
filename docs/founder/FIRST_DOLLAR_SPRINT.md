# First Dollar Sprint — Shipwright

Run this protocol if 5 days post-launch with 0 paying customers.

## 10 Direct Outreach Messages (Personalized)

Find 10 developers using this GitHub search:
```
https://github.com/search?q=stars%3A%3E3+pushed%3A%3C2025-01-01+NOT+deployed&type=repositories
```

Filter for repos with: decent stars, last commit >6 months ago, no website in description.

**Message template (DM on Twitter/X or GitHub):**
```
Hey [name] — I saw your [repo name] on GitHub. Looks like a solid [framework] project.

I built a tool called Shipwright that takes repos like yours and generates the deployment config, README, and landing page as a PR in about 30 seconds.

Would you try it on [repo name]? First generation is $5, Pro is $15/mo. I'll give you a free credit to try it — just reply here.

[YOUR URL]
```

**10 target archetypes to search for:**
1. Next.js portfolio sites never deployed (@nextjs created >2023 with 5+ stars, no website)
2. React component libraries without documentation
3. Node.js CLI tools with no README
4. Express APIs with no deployment config
5. Full-stack apps with TODO in README
6. Developer portfolio generators (meta — they built the tool but didn't ship it)
7. Productivity tools / habit trackers sitting on GitHub
8. AI wrappers / LLM tools from 2024 that were never productized
9. Game projects with no release
10. Web scraper tools with potential commercial value

## Activation Rate Goal
- 10 DMs sent → 3 replies → 1 purchase = first dollar
- If 0 purchases after 10 DMs: adjust pitch, send 20 more

## After First Purchase
1. Email the customer: "Thank you — you're customer #1. What repo did you use it on?"
2. Get a quote for the press kit
3. Tweet about it immediately (see Day 4 template in CONTENT_CALENDAR_30.md)
